# Sports-Management-Website-Project

### How to Run
1. create .env file
2. In it, put:
	MONGO_URI=mongodb_repo_link
	PORT=port(Normally 5000)

3. cd to folder with command prompt and enter 3 commands:
	- npm install mongodb (Only need if you have never used MongoDB, otherwise skip)
	- npm init -y
	- npm install express mongoose dotenv cors

4. run the server in command prompt:
	- node server.js

5. From here on, You'll need to open `http://localhost:PORT/` to start it.
